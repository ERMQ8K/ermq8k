What is Claude Code?

Claude Code is an AI coding assistant with Agents (autonomous helpers) and Skills (reusable workflows).

How to install:

  1. Go to claude.ai/code
  2. Sign up / Log in
  3. Download Claude Code app
  4. Install and open
  5. Authenticate your account

Step 2: Set Up Lovable

What is Lovable?

Lovable is an AI platform that converts designs to interactive prototypes.

How to set up:

  1. Go to lovable.dev
  2. Sign up (GitHub/Google)
  3. Create new project
  4. Connect Figma (optional, can do later)
  5. Explore interface
‚úÖ
Checkpoint: Can you create a new project?

Step 3: Configure Chakra UI MCP

What is MCP?

MCP = Model Context Protocol. It connects Claude to Chakra UI components, enabling AI-powered component generation.

Claude AI
‚Üí
MCP Server
‚Üí
Chakra UI

MCP Architecture: Claude communicates with Chakra UI through the Model Context Protocol

Configuration:

{
  "mcpServers": {
    "chakra-ui": {
      "command": "npx",
      "args": ["@chakra-ui/mcp-server"],
      "env": {}
    }
  }
}

üîò Button Components

Primary Button

Create a primary button component:
- Text: "Get Started"
- Background color: #0066FF (blue)
- Text color: #FFFFFF (white)
- Padding: 16px horizontal, 12px vertical
- Border radius: 8px
- Font size: 16px
- Font weight: 600

States:
- Default: Blue background (#0066FF)
- Hover: Darker blue (#0052CC)
- Active: Even darker (#003D99)
- Disabled: Gray (#CCCCCC), cursor not-allowed

Secondary Button

Create a secondary button component:
- Text: "Learn More"
- Background: Transparent
- Border: 2px solid #0066FF
- Text color: #0066FF
- Padding: 16px horizontal, 12px vertical
- Border radius: 8px

States:
- Default: Blue border, blue text
- Hover: Blue background, white text
- Active: Darker blue background
- Disabled: Gray border, gray text

üìù Input Components

Text Input

Create a text input component:
- Width: 100% (max 400px)
- Height: 48px
- Padding: 12px horizontal
- Border: 1px solid #CCCCCC
- Border radius: 8px
- Font size: 16px
- Placeholder: "Enter text..."

States:
- Default: Gray border
- Focus: Blue border (#0066FF), blue outline
- Error: Red border (#FF0000)
- Disabled: Gray background, gray border

üÉè Card Components

Basic Card

Create a card component:
- Background: #FFFFFF
- Border: 1px solid #E5E5E5
- Border radius: 12px
- Padding: 24px
- Shadow: 0 2px 8px rgba(0,0,0,0.1)

Content:
- Title (heading, 20px, bold)
- Description (body text, 16px, gray)
- Optional: Image at top
- Optional: Button at bottom

States:
- Default: Subtle shadow
- Hover: Larger shadow, slight lift
